# ML-based-Cancer-detection-in-liquid-biopsies

Analysis pipeline for transcription factor (TF) accessibility profiling from cell-free DNA (cfDNA) using nucleosome footprinting. Identifies differentially accessible TF binding sites across cancer types relative to healthy controls.

This pipeline builds on the Griffin framework for nucleosome profiling from cfDNA, which uses the characteristic footprint left by TF-bound nucleosomes to infer transcription factor activity at the genome-wide level. While the original Griffin study profiled a curated selection of TFs chosen for their relevance to specific cancer types, this analysis applies the same profiling approach to a substantially broader TF panel, enabling a less hypothesis-driven, more exploratory survey of TF accessibility changes across multiple cancer types relative to healthy controls.
